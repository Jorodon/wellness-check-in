## Getting Started
### 1. Initial Setup
* **Clone the repository:** Clone this project into VS Code.
* **Open Terminal:** Ensure the terminal path is at the project root (`.../wellness-check-in`).
### 2. Configure the Environment
* **Create a Virtual Environment:**
  ```powershell
  python -m venv venv
* **Activate the virtual environment:**
  ```powershell
  .\venv\Scripts\Activate.ps1
### 3. Install Dependencies
* **Install python packages:**
  ```powershell
  pip install -r requirements.txt
* **Install NPM packages:**
  ```powershell
  npm install
### 4. Select Python interpreter
* **Open Command Palette:** Hold CTRL+SHIFT+P and type/select the following.
  ```powershell
  Python: Select Interpreter
  ```
* **Choose Virtual Enironment:** From the menu that appears, select the interpreter with path:
  ```powershell
  .\venv\Scripts\python.exe
### 5. Generate CSS
* **Two options available:**
    * One time build:
    ```powershell
    npm run build
    ```
    * Continuous build, updates as code changes:
    ```powershell
    npm run watch
    ```
### 6. Run web app
* **Start web app:**
  ```powershell
  python checkin_backend\run.py
* **Navigate to website:** click on the link in terminal, or paste "http://127.0.0.1:5000" into browser URL.
